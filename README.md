# Comet X Ultimate v4

إضافة كروم ذكية مع لوحة جانبية (Side Panel) وتكامل مع GitHub.

## المميزات
- **Side Panel**: لوحة جانبية دائمة.
- **GitHub Import**: جلب Issues و Pull Requests من مستودعك الخاص.
- **ملاحظات**: كتابة وحفظ ملاحظات سريعة.
- **دعم عربي**: واجهة عربية كاملة (RTL).

## طريقة التثبيت
1. افتح `chrome://extensions` في متصفح كروم.
2. فعّل **Developer mode** (الزاوية العليا اليمنى).
3. اضغط **Load unpacked**.
4. اختر مجلد `cometx-ultimate-v4`.

## الإعداد
1. اضغط على أيقونة الإضافة لفتح اللوحة الجانبية.
2. اضغط على زر الإعدادات (⚙️).
3. أدخل GitHub Token و اسم المستودع (مثال: `owner/repo`).
4. احفظ الإعدادات.
